ORCINY KICAD NET MAP (apply to orciny_device.kicad_sch)
=======================================================

Project files
- orciny_device.kicad_pro
- orciny_device.kicad_sch

Purpose
- This KiCad project is an openable starter.
- Replace template symbols with your preferred parts and wire according to this net map.

Controller-to-controller link
- RP2040_SERIAL1_TX  -> M0_SERIAL1_RX
- RP2040_GND         -> M0_GND

Switch controls (momentary switches to GND, firmware uses INPUT_PULLUP)
- SW1: RP2040_PIN2 -> switch -> GND  (toggle on/off)
- SW2: RP2040_PIN3 -> switch -> GND  (previous sequence, 1->3 wrap)
- SW3: RP2040_PIN4 -> switch -> GND  (next sequence, 3->1 wrap)
- Hold SW1+SW3 for 5 seconds -> reset to sequence 1

RP2040 effect outputs (all high-current paths require MOSFET/driver stages)
- RP2040_PIN5  -> SPARK1_GATE
- RP2040_PIN6  -> SPARK2_GATE
- RP2040_PIN9  -> SPARK3_GATE
- RP2040_PIN10 -> SPARK4_GATE
- RP2040_PIN11 -> PULSE_FILAMENT_GATE
- RP2040_PIN12 -> BEAM_R_GATE
- RP2040_PIN13 -> BEAM_G_GATE
- RP2040_PIN14 -> BEAM_B_GATE
- RP2040_PIN15 -> PELTIER_ENABLE_GATE

Motor FeatherWing section
- RP2040_SDA -> MOTOR_WING_SDA
- RP2040_SCL -> MOTOR_WING_SCL
- MOTOR_PORT_M1_A/B -> DC_PUMP
- STEPPER_PORT2_M3/M4 -> CLAW_STEPPER

M0 + NeoPXL8 core outputs
- NEOPXL8_D0 -> CORE_STRIP_1_DATA
- NEOPXL8_D1 -> CORE_STRIP_2_DATA
- NEOPXL8_D2 -> CORE_STRIP_3_DATA
- NEOPXL8_D3 -> CORE_STRIP_4_DATA

Power domains
- +12V_MAIN -> Motor wing VM, pump supply, stepper supply, peltier power stage
- +5V_LOGIC -> RP2040, M0, NeoPXL8 logic, NeoPixel strip power (if 5V strips)
- GND_COMMON -> all grounds tied together

KiCad checklist
1) Open orciny_device.kicad_pro in KiCad.
2) In Schematic Editor, replace template Arduino symbols with your board symbols or generic headers.
3) Add named net labels exactly as above.
4) Add MOSFET symbols for each high-current channel and wire gate/source/drain appropriately.
5) Run ERC and resolve only intentional warnings.
6) Save as a new revision if you want to preserve the starter state.

Concrete Adafruit_Eagle footprint mapping (project-local)
- Library: Adafruit_Eagle (from fp-lib-table)

Primary boards and wings
- Feather RP2040 controller placement: Adafruit_Eagle:FEATHERWING
- Feather M0 controller placement: Adafruit_Eagle:FEATHERWING
- Motor FeatherWing placement: Adafruit_Eagle:FEATHERWING
- NeoPXL8 FeatherWing placement: Adafruit_Eagle:FEATHERWING
- Feather Doubler carrier: Adafruit_Eagle:FEATHERWING_DIM

Switches
- SW1, SW2, SW3 (momentary): Adafruit_Eagle:PUSHBUTTON_SMD_SJ

Power and high-current staging
- 12V input jack: Adafruit_Eagle:DCJACK_2MM_PTH
- Pump / peltier / external load connectors (2-pin JST option): Adafruit_Eagle:JST-PH-2-THM
- Stepper or beam/power screw terminal option (4-pin): Adafruit_Eagle:1X04-3.5MM
- Through-hole TO-220 MOSFET stages: Adafruit_Eagle:TO220

Header options for custom daughter wiring
- 1x03 signal headers: Adafruit_Eagle:1X03
- 1x04 signal headers: Adafruit_Eagle:1X04-BIG
- 1x06 signal headers: Adafruit_Eagle:1X06
