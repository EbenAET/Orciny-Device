ORCINY DEVICE - ASCII CIRCUIT DIAGRAM
======================================
Rev D  (single-strand NeoPixel; direct GPIO pump control)

Power:
  5V battery (J801)  -> main fused input -> +5V_MAIN trunk
  +5V_MAIN trunk     -+-> F801 (direct) -------- +5V_BUS rail (RP2040, M0 power; NeoPixel power; load supply)
                      +-> F802 -> +3V_FILAMENT buck -> +3V_FILAMENT rail (filaments Q1-Q5 drains only)

  +5V_BUS rail       -> RP2040 (internal LDO -> 3.3V logic), M0 (internal LDO -> 3.3V logic),
                        NeoPixel strip power, beam/peltier/servo/pump supply
  +3V_FILAMENT rail  -> Spark/pulse filament MOSFET drains (Q1-Q5)
  GND_COMMON         -> COMMON to every board, switch, driver, and load

 Switches (momentary, wired to GND; firmware uses INPUT_PULLUP):
 ┌────────┐        ┌────────┐        ┌────────┐
 │  SW1   │        │  SW2   │        │  SW3   │
 │ On/Off │        │  Prev  │        │  Next  │
 └───┬────┘        └───┬────┘        └───┬────┘
     |                 |                 |
   M0 D2             M0 D3             M0 D4
   GND               GND               GND
   Hold SW1 + SW3 together for 5s -> reset to sequence 1


 MOSFET Driver Stage (each high-current channel):

   RP2040 GPIO pin
         |
     (gate)  N-channel MOSFET (logic-level, Vgs(th) < 2.5V)
         |
   drain -- Load negative terminal
   source -- GND_COMMON

  Load positive terminal -- +3V_FILAMENT (spark/pulse)
                  or +5V_BUS      (beam/peltier/pump)


 Filament Channels (J601 -> R601-R605 -> J603):

   +3V_FILAMENT -> J602 power input
   J601 PWM bus -> R601 10Ω -> Spark Filament 1 load
   J601 PWM bus -> R602 10Ω -> Spark Filament 2 load
   J601 PWM bus -> R603 10Ω -> Spark Filament 3 load
   J601 PWM bus -> R604 10Ω -> Spark Filament 4 load
   J601 PWM bus -> R605 10Ω -> Pulse Filament load


 Beam RGB Channels (J701 -> R701-R703 -> J702):

  +5V_BUS -> J701 power input (for high-power RGB LED)
   J701 PWM inputs (R/G/B from RP2040 GP12/13/14)
   -> R701 10Ω -> Beam Red   output (J702 pin 1)
   -> R702 10Ω -> Beam Green output (J702 pin 2)
   -> R703 10Ω -> Beam Blue  output (J702 pin 3)
   J702 pin 4 -> GND_COMMON


 Power Domain Summary:
   Rail              Source               Protected By   Consumers
   +5V_BUS           Direct from 5V bat   F801 (3A)      RP2040 (with internal LDO), M0 (with internal LDO),
                                                         NeoPixel strip, beam RGB, peltier, servo, pump
   +3V_FILAMENT      3V buck from 5V      F802 (2A)      Spark/pulse filament MOSFET drains (Q1-Q5)


 Quick Net Summary:
   M0 D2     <- SW1 -> GND   (toggle ON/OFF)
   M0 D3     <- SW2 -> GND   (previous sequence, 1->3 wrap)
   M0 D4     <- SW3 -> GND   (next sequence, 3->1 wrap)
   M0 D13    -> NeoPixel strip data (30 pixels, WS2812B, single strand)
   M0 TX     -> RP2040 RX    (40ms EffectCommand frames)
   RP2040 GP5-GP11  -> MOSFET gates for filament channels (Q1-Q5)
   RP2040 GP12-GP14 -> MOSFET gates for beam RGB channels (Q6-Q8)
   RP2040 GP15      -> MOSFET/relay gate for peltier (Q9)
   RP2040 GP8       -> MOSFET gate for pump control (Q10)
   RP2040 GP7       -> FP-S14B servo signal
   Hold SW1 + SW3 for 5s => reset to sequence 1
