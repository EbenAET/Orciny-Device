ORCINY DEVICE - EXAMPLE ASCII CIRCUIT DIAGRAM
=============================================

Power:
  +12V main rail     -> Beam/Spark/Pulse driver supply, Peltier branch, Motor Wing VM input
  +5V logic buck     -> RP2040 Feather, M0 Feather, NeoPXL8 logic, NeoPixel strip power
  +5V actuator buck  -> FP-S14B servo power, 5V peristaltic pump branch
  GND                -> COMMON to every board, switch, driver, and load

                           |     FX Executor Node      |
                           |                           |
Spark Filament 1 driver <--| GP5   PWM                 |
Spark Filament 2 driver <--| GP6   PWM                 |
Spark Filament 3 driver <--| GP9   PWM                 |

                           |        FEATHER M0         |
                           |   Primary Scene Control   |
                           |                           |
Switch 1 (On/Off) ---------| D2   (INPUT_PULLUP)       |

                           |                           |
NeoPXL8 outputs D0..D3 --->| 4 Core Strip Sets         |
(remaining D4..D7 unused)  |                           |
                           +---------------------------+


NeoPXL8 / Core Strips:
  D0 -> Core Strip Set 1
  D1 -> Core Strip Set 2
  D2 -> Core Strip Set 3
  D3 -> Core Strip Set 4


Driver Stages (required for high current loads):
  RP2040 GPIO -> Gate resistor -> MOSFET gate
  MOSFET source -> GND
  MOSFET drain  -> Load negative
  Load positive -> +V rail (5V or 12V per device)

Current-limiting resistor rules:
  Spark1..4 and Pulse channels: 10 ohm series resistor per LED load leg
  Beam R/G/B channels: 1 ohm series resistor per color leg


Grounding Rules:
  [MANDATORY] Tie all grounds together:
    - RP2040 ground
    - M0 ground
    - Motor FeatherWing ground
    - NeoPXL8 / strip ground
    - Pump/stepper/beam/peltier driver grounds
    - Switch grounds


Quick Net Summary:
  M0 D2       <- SW1 to GND  (toggle on/off)
  M0 D3       <- SW2 to GND  (previous sequence, 1->3 wrap)
  M0 D4       <- SW3 to GND  (next sequence, 3->1 wrap)
  Hold SW1 + SW3 for 5s => reset sequence to 1
  M0 TX       -> RP2040 RX   (FX command frames)
  RP2040 GP7  -> FP-S14B signal
